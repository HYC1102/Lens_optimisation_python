Ray tracing python object oriented programming project: Created by Choy Hong Yau (hyc219@ic.ac.uk)

(CID 01704380)

The entire project contains 4 modules:

1) Vector func
2) raytracer
3) OpticalElement
4) InvestigationTools

Please run the above 4 classes prior to running the scripts.

The file also contains 2 scripts:

1) Module testing - Shows preliminary testings before investigation. Please unhash lines to see results. 
2) Investigations - Shows the investigations carried out using the ray tracer, also please unhash lines to see full investigations.
